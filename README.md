# Bank Account Management System

## Description
This project is a Python-based bank account system that allows users to deposit, withdraw and view account details.

## Features
- Deposit money
- Withdraw money
- Balance validation
- Transaction history tracking
- Savings account with interest

## OOP Concepts Used
- Encapsulation
- Inheritance
- Properties
- Method overriding

## How to Run
1. Open the file bank_account.py
2. Run using Python
3. View output in terminal
